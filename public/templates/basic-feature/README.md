# CDAT Feature Template

Generated from https://cdat.sdet.it/resources

## Usage

1. Drop this folder into your `features/` directory
2. Rename `Feature` to your feature name (e.g., `Login`, `Cart`, `Checkout`)
3. Drop the `.template` suffix from the four files
4. Implement each layer
